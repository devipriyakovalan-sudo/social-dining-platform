# 🛍️ ShopEasy – Online Shopping Website

A fully responsive e-commerce front-end with product browsing, category filtering, search, cart management, and checkout flow — built with pure HTML, CSS and JavaScript.

## 🚀 Live Demo
> Open `index.html` in your browser — no server setup needed!

## 📸 Features

- **Product Catalogue** — 6 products across 4 categories with prices and discounts
- **Category Filter** — Filter by Electronics, Fashion, Home & Kitchen, Books
- **Live Search** — Real-time search from the navbar
- **Shopping Cart** — Add/remove items, auto-updates total and item count
- **Checkout Flow** — Cart summary with free delivery and order confirmation
- **Responsive Design** — Mobile, tablet and desktop friendly

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Semantic page structure |
| CSS3 | Flexbox, Grid, transitions, responsive breakpoints |
| JavaScript (Vanilla) | Cart state management, filtering, live search, dynamic rendering |

## 📁 Project Structure

```
online-shopping/
├── index.html     # Full page structure (navbar, hero, products, cart)
├── style.css      # Complete styling with responsive design
├── app.js         # Cart logic, filter, search, checkout
└── README.md      # Project documentation
```

## 🏃 How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/devipriyakovalan/online-shopping-website.git
   ```
2. Open `index.html` in any browser
3. No dependencies or installation needed!

## 💡 Key Concepts Demonstrated

- State management using JavaScript arrays (cart state)
- Dynamic DOM updates without any framework
- CSS Grid for responsive product layouts
- CSS transitions and hover effects
- Event-driven UI interactions
